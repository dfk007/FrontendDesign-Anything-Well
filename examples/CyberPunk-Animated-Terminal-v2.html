// Runbook

How to operate the checklist. Press Enter in the input to commit a new task.

1

Initialize Session

State hydrates from localStorage under key todo-checklist-state. If absent, the default seed tasks are loaded.

2

Toggle Task Status

Click the neon checkbox on any row to flip done/pending. The progress meter and counts recompute instantly.

3

Commit New Task

Type into the input under the task list, then press Enter or click EXECUTE. The new task appends as Pending.

4

Remove Task

Press REMOVE on a row to purge it. The localStorage mirror updates immediately.
